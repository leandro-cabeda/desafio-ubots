# 1º Etapa Instalar todas dependecias do projeto
## Comando
npm i


# 2º Etapa
## Instalar o nodemon global ou não

npm i nodemon

# 3º Etapa
npm start

## Com isso vai estar já pronto e executando o projeto e depois só acessar o link:
localhost:3000

- Feito!
